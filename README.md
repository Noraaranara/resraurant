✨ Описание проекта: "Online restaurant website":
Цель проекта — разработать современный веб‑сайт для ресторана, который отразит концепцию и атмосферу заведения и обеспечит удобство для гостей при ознакомлении с меню и бронированием.

✨ Запуск проекта:
1. Клонируйте репозиторий:
   git clone https://github.com/имя/проект.git

2. Перейдите в папку проекта
    cd проект

3. Откройте файл index.html в браузере.

✨ Использованные технологии:
HTML5, CSS (SASS), JavaScript, БЭМ.
